Angular Universal Addon allow you to enable Server Side Rendering (SSR) on your angular application

Advantages

  1. Facilitate web crawlers (SEO)
  2. Improve performance on mobile and low-powered devices
  3. Show the first page quickly

To turn a regular angular application to a Universal App you will need to

  • Install @angular/platform-server
  • a server-side app module, app.server.module.ts
  • an entry point for the server-side, main.server.ts
  • a TypeScript config file, tsconfig.server.json
  • Modify your angular.json config files
  • Build your browser-side and server-side angular sources in AOT
